Synctrip

A real time, social trip planner and visualization tool.

View project onGitHub

Motivation

This project was motivated by my interest in learning some new technologies: AngularJS, Firebase, and Cordova. Along the way, I also got to play with some other services like Google Maps API and (future work) some sort of image host (flickr? instagram? s3?). The code is private for now, until I decide if the project is worthy of app stores.
